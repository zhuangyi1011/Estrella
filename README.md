
# 大数据课题组综合管理制度
## 一 、课题组织架构
### 1. 管理模式
大数据课题组采用矩阵式管理，通过横向和纵向的管理模式，提升课题组的工作效率，打破部门壁垒，达到人员的充分调配。
### 2. 团队设置
大数据课题组，下设多个工作小组，分别为：
产品组、JAVA一组（平台方向）、JAVA二组（业务方向）、前端一组（平台方向）、前端二组（业务方向）、大数据组、可视化组、UI组、数据处理组、运维组、测试组、技术支持组和管理后勤组。
### 3. 协作模式
大数据课题组以产品为主线，协调各组人力资源，在开发完成后，人力资源回归本小组进行下一次的分配。
